| Symbol | Meaning      | Example        |         |         |
| ------ | ------------ | -------------- | ------- | ------- |
| `+`    | At least 1   | `(student+)`   |         |         |
| `*`    | Zero or more | `(student*)`   |         |         |
| `?`    | Optional     | `(student?)`   |         |         |
| `,`    | Sequence     | `(roll, name)` |         |         |
| `      | `            | Choice         | `(email | phone)` |
